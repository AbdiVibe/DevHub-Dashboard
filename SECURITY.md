## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue, please follow these steps:

1. **Do not** disclose the vulnerability publicly
2. Email us at: your-email@example.com
3. Include as much detail as possible:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

## What to expect

- You will receive acknowledgment of your report within 48 hours
- We will investigate and provide status updates
- Once resolved, we will credit you in release notes

## Security Best Practices

- Always use HTTPS
- Keep all dependencies updated
- Sanitize user input
- Use secure API keys (never commit to git)

## Data Privacy

This application stores all data locally in your browser's localStorage.
No data is transmitted to any server except:
- Weather API requests to OpenWeatherMap
- GitHub API requests for user data
